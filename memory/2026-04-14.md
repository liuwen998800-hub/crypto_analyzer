### 2. 下单逻辑最终版本
```
每周期：
1. 取消所有未成交订单 (polymarket clob cancel-all)
2. 查询订单簿获取最优价格
3. 优先市价单(FAK) --amount $2
4. 市价单失败 → 限价单(GTC) @ 最优价 × 0.98
5. 限价单价格取2位小数 (tick size = 0.01)
```

### 8. 三信号确认机制
```
信号1: 技术指标评分 → YES/NO/HOLD
信号2: AI分析(置信度≥55%) → YES/HOLD
信号3: 币安价差(±0.3%) → YES/NO/HOLD
       - 币安BTC涨价 >+0.3% → YES
       - 币安BTC跌价 <-0.3% → NO
       - 其他 → HOLD

入场条件：三信号中有两者一致即入场
```

### 9. Kelly仓位管理
```python
KELLY_FRACTION = 0.25  # 使用1/4 Kelly

公式: f = (b×p - q) / b × 0.25
- b = 盈亏比 (默认1.5)
- p = 胜率 (默认0.55)
- q = 1-p

示例: 钱包$36 → 建议仓位$2.25 (6.3%)
最低仓位: $2.00
```
